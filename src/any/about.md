# モジュール `std::vec`

## バージョン

Rust 1.0.0 ~

## 概要

動的型付けと型リフレクションに関する機能を提供する。

## 構造体

| 名前 | 説明 |
| --- | --- |
| `TypeId` | グローバルに一意な型の識別子を提供する |

## トレイト

| 名前 | 説明 |
| --- | --- |
| `Any` | 動的型付けを模倣する |

## 関数

| 名前 | 説明 |
| --- | --- |
| `type_name` | ジェネリクスに渡された型の名前を取得する |
| `type_name_of_val` | 引数に渡された値の方の名前を取得する |
